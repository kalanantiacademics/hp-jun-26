Lokasi Script di Roblox Studio

Setiap script punya tempat khususnya masing-masing di Explorer.

Tempat menaruh script yang mengatur seluruh jalannya game, seperti skor dan pemain masuk.

🌍 Workspace (Dalam Objek)

▶️ Script 2 & 3: PickUp & Sorting

1. Membuat Leaderboard

Pergi ke ServerScriptService (di kanan layer Explorer), klik + lalu tambahkan Script. Ganti namanya jadi leaderboard.

ServerScriptService > leaderboard
local Players = game:GetService("Players") Players.PlayerAdded:Connect(function(player) local leaderstats = Instance.new("Folder") leaderstats.Name = "leaderstats" leaderstats.Parent = player local items = Instance.new("IntValue") items.Name = "Organic" items.Value = 0 items.Parent = leaderstats local items2 = Instance.new("IntValue") items2.Name = "Non Organic" items2.Value = 0 items2.Parent = leaderstats end)

Fungsi Leaderboard Script

Script ini berfungsi membuat buku catatan poin di pojok kanan atas layar setiap ada pemain yang masuk.

Di dalam script tadi, kita membuat dua jenis skor:

  1. Skor untuk item "Organic"
  2. Skor untuk item "Non Organic"
PENTING: Kamu bisa dan HARUS mengubah teks "Organic" dan "Non Organic" di script sesuai dengan kategori barang di game-mu. Misalnya: "Barang Sehat", "Buku", dll.

2. Membuat Pick Up Script

Pilih salah satu item/Tool kamu di Workspace. Tambahkan Script di dalam Tool tersebut, beri nama PickUpScript.

Workspace > (Nama Tool Kamu) > PickUpScript
local Players = game:GetService("Players") local tool = script.Parent tool.Equipped:Connect(function() local character = tool.Parent local player = Players:GetPlayerFromCharacter(character) if player then tool:SetAttribute("OwnerUserId", player.UserId) print("Owner saved:", player.Name, player.UserId) end end)

*Ingat: Script ini harus di-copy/paste ke dalam SEMUA item yang bisa diambil!

Fungsi PickUpScript

Script ini menempel pada setiap Tool (barang).

Ketika pemain menyentuh atau mengambil barang tersebut masuk ke tas, script ini langsung memberikan tanda pengenal (ID) pemain pada barang itu.

Tujuannya? Agar Roblox tahu barang tersebut adalah "milik" pemain yang mana.

3. Membuat Sorting Script

Tambahkan Script ke dalam objek Dropping Place (Part) kamu, beri nama SortingScript.

Workspace > (Nama Bucket/Part) > SortingScript
local Players = game:GetService("Players") -- UBAH DAFTAR NAMA INI SESUAI NAMA ITEM-MU! local organicItemList = {"leaf", "potato", "banana"} script.Parent.Touched:Connect(function(object) local tool = object:FindFirstAncestorOfClass("Tool") if tool and table.find(organicItemList, tool.Name) then local ownerId = tool:GetAttribute("OwnerUserId") local player = Players:GetPlayerByUserId(ownerId) if player then local leaderstats = player:FindFirstChild("leaderstats") local items = leaderstats and leaderstats:FindFirstChild("Organic") if items then items.Value += 1 end end tool:Destroy() else if tool then local ownerId = tool:GetAttribute("OwnerUserId") if ownerId then local player = Players:GetPlayerByUserId(ownerId) if player then local backpack = player:FindFirstChild("Backpack") if backpack then tool.Parent = backpack end end end end end end)

Memahami Sorting Script

Ini adalah "juri" yang menentukan apakah pemain memasukkan item ke tempat yang benar.

✅ Jika Item Benar

Script akan mengecek kecocokan nama item. Jika sesuai, script akan menambahkan skor pemain +1, dan langsung menghancurkan item (`tool:Destroy()`) agar tidak menumpuk.

❌ Jika Item Salah

Jika pemain menjatuhkan item yang salah ke bucket, item tersebut tidak akan hilang. Script akan mengembalikan item tersebut secara paksa ke dalam tas (`Backpack`) pemain.

🚨 PENTING: Perhatikan Baris ke-4!
local organicItemList = {"leaf", "potato", "banana"}

Kata "leaf", "potato", dan "banana" WAJIB kamu ganti agar sama persis dengan nama-nama item (Tool) yang ada di Workspace-mu. Pastikan huruf besar-kecilnya sama!
